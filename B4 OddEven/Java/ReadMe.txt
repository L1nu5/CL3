1. Install Apache Tomcat 6.0
2. Download Servlet-api.jar & put it in OddEven/src
3. Copy the OddEven folder to Tomcat/WebApps/
4. You may need to compile the Sort.java (OddEven/src) 
	javac -cp servlet-api.jar Sort.java
5. Restart Apache Tomcat
6. To run the code, Type localhost:port/OddEven/index.html

Note: Admin/Superuser login required.
